# 📝 Todo List

A modern Todo List application built using **HTML, CSS, and JavaScript**.

## 🚀 Live Demo

https://somnathbuilds.github.io/todo-list/

## 📸 Screenshot

![Todo List](assets/screenshot.png)

## ✨ Features

- ✅ Add Task
- ✏️ Edit Task
- 🗑️ Delete Task
- ✔️ Mark Task as Complete
- 💾 Local Storage
- 📊 Task Counter
- 🧹 Clear All Tasks
- ⌨️ Add task using Enter key
- 📱 Responsive Design

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
